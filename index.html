About this project

This repository contains the static HTML and CSS source for a small class project. If your site has a styles.css file at the repository root, it will be used automatically. Otherwise the page falls back to simple built-in styles so you can preview the content immediately.

If you already have an index.html in the repo root with your own markup, keep that file and remove this one — GitHub Pages serves the file named index.html at the site root.
